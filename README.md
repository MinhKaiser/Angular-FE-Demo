Angular 21 training application backed by the DummyJSON API. The project demonstrates
feature-first organization, standalone components, Signal Store, one-way data flow, lazy
routes, functional interceptors, strict TypeScript, and reusable presentational components.

## Requirements

- Node.js `^20.19.0`, `^22.12.0`, or `>=24.0.0`
- npm `>=8`

## Commands

```bash
npm ci
npm start
npm run typecheck
npm test
npm run format:check
npm run build:production
```
